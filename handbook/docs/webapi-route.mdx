## 一、路由规则

TouchSocket.WebApi 提供灵活的路由系统，支持多种路由配置方式。

<BilibiliCard title="WebApi路由" link="https://www.bilibili.com/cheese/play/ep1688330" isPro="true"/>

### 1.1 默认路由

如果不指定路由，系统会自动生成默认路由：路由格式为：`/{服务类名}/{方法名}`（不区分大小写）

```
/{ClassName}/{MethodName}
```

示例：

<CustomCodeBlock region="定义WebApi服务"/>

访问路径：`/demoApiServer/sum?a=10&b=20`

### 1.2 使用 Router 特性

可以使用 `[Router]` 特性自定义路由规则，支持使用占位符：

- `[api]`：表示服务类名
- `[action]`：表示方法名

<CustomCodeBlock region="Router路由"/>

### 1.3 多路由支持

一个方法可以配置多个路由：

```csharp showLineNumbers
public partial class ApiServer : SingletonRpcServer
{
    [Router("[api]/[action]")]
    [Router("sum")]
    [Router("calculate/add")]
    [WebApi(Method = HttpMethodType.Get)]
    public int Sum(int a, int b)
    {
        return a + b;
    }
}
```

可以通过以下任意路径访问：
- `/apiserver/sum?a=10&b=20`
- `/sum?a=10&b=20`
- `/calculate/add?a=10&b=20`

### 1.4 正则路由

支持使用正则表达式匹配路由：

```csharp showLineNumbers
public partial class ApiServer : SingletonRpcServer
{
    [Router(@"^/users/\d+$")]
    [WebApi(Method = HttpMethodType.Get)]
    public string GetUserById()
    {
        // 从路径中提取ID
        return "User info";
    }
}
```

### 1.5 路由匹配规则

1. **不区分大小写**：`/ApiServer/Sum` 和 `/apiserver/sum` 等效
2. **优先级**：精确匹配 > 正则匹配 > 默认路由
3. **路径规范化**：自动移除前后斜杠

## 二、完整示例

```csharp showLineNumbers
[Router("[api]")]
public partial class UserController : SingletonRpcServer
{
    // GET /user/list
    [WebApi(Method = HttpMethodType.Get)]
    public List<string> List()
    {
        return new List<string> { "Alice", "Bob", "Charlie" };
    }

    // GET /user/1 或 /api/user/detail?id=1
    [Router("[api]/{id}")]
    [Router("[api]/detail")]
    [WebApi(Method = HttpMethodType.Get)]
    public string GetById(int id)
    {
        return $"User {id} details";
    }

    // POST /user/create
    [WebApi(Method = HttpMethodType.Post)]
    public string Create([FromBody] UserDto user)
    {
        return $"Created user: {user.Name}";
    }

    // PUT /user/1
    [Router("[api]/{id}")]
    [WebApi(Method = HttpMethodType.Put)]
    public string Update(int id, [FromBody] UserDto user)
    {
        return $"Updated user {id}: {user.Name}";
    }

    // DELETE /user/1
    [Router("[api]/{id}")]
    [WebApi(Method = HttpMethodType.Delete)]
    public string Delete(int id)
    {
        return $"Deleted user {id}";
    }
}

public class UserDto
{
    public string Name { get; set; }
    public int Age { get; set; }
}
```
