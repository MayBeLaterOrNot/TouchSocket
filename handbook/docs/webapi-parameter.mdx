## 一、参数绑定

TouchSocket.WebApi 支持从多个来源获取参数。

<BilibiliCard title="参数获取" link="https://www.bilibili.com/cheese/play/ep1688331" isPro="true"/>

### 1.1 GET 请求参数

GET 请求默认从 **Query String** 获取参数：

```csharp showLineNumbers
[WebApi(Method = HttpMethodType.Get)]
public int Sum(int a, int b)
{
    return a + b;
}
```

访问：`/apiserver/sum?a=10&b=20`

### 1.2 POST 请求参数

POST 请求的参数获取规则：

**规则1：简单类型从 Query**

```csharp showLineNumbers
[WebApi(Method = HttpMethodType.Post)]
public string SayHello(string name)
{
    return $"Hello, {name}!";
}
```

访问：`POST /apiserver/sayhello?name=Tom`

**规则2：复杂类型从 Body**

```csharp showLineNumbers
public class LoginDto
{
    public string Username { get; set; }
    public string Password { get; set; }
}

[WebApi(Method = HttpMethodType.Post)]
public string Login(LoginDto dto)
{
    return $"Welcome, {dto.Username}!";
}
```

请求：
```json
POST /apiserver/login
Content-Type: application/json

{
  "username": "admin",
  "password": "123456"
}
```

### 1.3 FromQuery - 从查询字符串

显式指定从 Query String 获取：

```csharp showLineNumbers
[WebApi(Method = HttpMethodType.Post)]
public string Search([FromQuery] string keyword, [FromBody] FilterDto filter)
{
    return $"Searching for: {keyword}";
}
```

访问：`POST /apiserver/search?keyword=test`

### 1.4 FromHeader - 从请求头

从 HTTP Header 获取参数：

```csharp showLineNumbers
[WebApi(Method = HttpMethodType.Get)]
public string GetWithAuth([FromHeader] string authorization)
{
    return $"Token: {authorization}";
}
```

请求：
```
GET /apiserver/getwithauth
Authorization: Bearer token123
```

### 1.5 FromForm - 从表单

从表单数据获取参数：

```csharp showLineNumbers
[WebApi(Method = HttpMethodType.Post)]
public string SubmitForm([FromForm] string username, [FromForm] string email)
{
    return $"User: {username}, Email: {email}";
}
```

请求：
```
POST /apiserver/submitform
Content-Type: application/x-www-form-urlencoded

username=Tom&email=tom@example.com
```

### 1.6 FromBody - 从请求体

显式指定从 Body 获取（JSON）：

```csharp showLineNumbers
public class UserDto
{
    public string Name { get; set; }
    public int Age { get; set; }
}

[WebApi(Method = HttpMethodType.Post)]
public string CreateUser([FromBody] UserDto user)
{
    return $"Created: {user.Name}, Age: {user.Age}";
}
```

请求：
```json
POST /apiserver/createuser
Content-Type: application/json

{
  "name": "Alice",
  "age": 25
}
```

### 1.7 混合参数

支持混合使用多种参数来源：

```csharp showLineNumbers
[WebApi(Method = HttpMethodType.Post)]
public string MixedParams(
    [FromQuery] int id,
    [FromHeader] string authorization,
    [FromBody] UserDto user)
{
    return $"ID: {id}, User: {user.Name}, Auth: {authorization}";
}
```

请求：
```
POST /apiserver/mixedparams?id=100
Authorization: Bearer token123
Content-Type: application/json

{
  "name": "Bob",
  "age": 30
}
```

## 二、参数类型支持

### 2.1 基础类型

支持所有 C# 基础类型：
- `int`, `long`, `double`, `decimal`, `bool`
- `string`, `DateTime`, `Guid`
- 可空类型：`int?`, `DateTime?` 等

### 2.2 集合类型

支持数组和集合：

```csharp showLineNumbers
[WebApi(Method = HttpMethodType.Get)]
public int SumArray([FromQuery] int[] numbers)
{
    return numbers.Sum();
}
```

访问：`/apiserver/sumarray?numbers=1&numbers=2&numbers=3`

### 2.3 复杂对象

支持嵌套对象：

```csharp showLineNumbers
public class OrderDto
{
    public int OrderId { get; set; }
    public List<OrderItem> Items { get; set; }
}

public class OrderItem
{
    public string ProductName { get; set; }
    public int Quantity { get; set; }
}

[WebApi(Method = HttpMethodType.Post)]
public string CreateOrder([FromBody] OrderDto order)
{
    return $"Order {order.OrderId} with {order.Items.Count} items";
}
```

## 三、完整示例

```csharp showLineNumbers
public partial class UserApiServer : SingletonRpcServer
{
    // GET 请求 - 简单参数
    [WebApi(Method = HttpMethodType.Get)]
    public List<string> Search(string keyword, int page = 1, int size = 10)
    {
        return new List<string> { $"Results for '{keyword}' (page {page})" };
    }

    // POST 请求 - Body 参数
    [WebApi(Method = HttpMethodType.Post)]
    public string Register([FromBody] UserDto user)
    {
        return $"Registered: {user.Name}";
    }

    // POST 请求 - 混合参数
    [WebApi(Method = HttpMethodType.Post)]
    public string UpdateProfile(
        [FromQuery] int userId,
        [FromHeader(Name = "X-API-Key")] string apiKey,
        [FromBody] UserDto user)
    {
        return $"Updated user {userId}";
    }

    // POST 请求 - 表单数据
    [WebApi(Method = HttpMethodType.Post)]
    public string Login([FromForm] string username, [FromForm] string password)
    {
        if (username == "admin" && password == "123456")
        {
            return "Login successful";
        }
        return "Invalid credentials";
    }
}
```
