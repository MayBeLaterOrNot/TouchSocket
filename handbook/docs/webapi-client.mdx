## 二、浏览器调用

最简单的方式是直接通过浏览器访问：

```
http://localhost:7789/apiserver/sum?a=10&b=20
```

使用 JavaScript/Fetch API：

```javascript
// GET 请求
fetch('http://localhost:7789/apiserver/sum?a=10&b=20')
  .then(response => response.json())
  .then(data => console.log(data));

// POST 请求
fetch('http://localhost:7789/apiserver/create', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({ name: 'Alice', age: 25 })
})
  .then(response => response.json())
  .then(data => console.log(data));
```

## 三、使用标准 HttpClient

### 3.1 GET 请求

```csharp
using var httpClient = new HttpClient();
var result = await httpClient.GetStringAsync("http://localhost:7789/apiserver/sum?a=10&b=20");
Console.WriteLine(result); // 输出: 30
```

### 3.2 POST 请求

```csharp
using var httpClient = new HttpClient();
var content = new StringContent(
    JsonConvert.SerializeObject(new { name = "Alice", age = 25 }),
    Encoding.UTF8,
    "application/json");

var response = await httpClient.PostAsync("http://localhost:7789/apiserver/create", content);
var result = await response.Content.ReadAsStringAsync();
Console.WriteLine(result);
```

## 六、使用 DispatchProxy 代理

使用代理可以像调用本地方法一样调用远程 API。

<BilibiliCard title="代码生成" link="https://www.bilibili.com/cheese/play/ep1688336" isPro="true"/>

### 6.1 定义接口

```csharp
public interface IApiServer
{
    [WebApi(Method = HttpMethodType.Get)]
    Task<int> Sum(int a, int b);

    [WebApi(Method = HttpMethodType.Post)]
    Task<string> CreateUser([FromBody] UserDto user);
}
```

### 6.2 创建代理

```csharp
var client = new HttpClient();
client.BaseAddress = new Uri("http://localhost:7789");

var proxy = client.GetWebApiDispatchProxy<IApiServer>();

// 调用方法
var result = await proxy.Sum(10, 20);
Console.WriteLine(result); // 输出: 30

var user = new UserDto { Name = "Alice", Age = 25 };
var createResult = await proxy.CreateUser(user);
```

## 七、代码生成

TouchSocket 支持通过源生成器或代码生成自动生成客户端代理。

### 7.1 服务端代码生成

在服务端注册服务时生成客户端代理代码：

```csharp
a.AddRpcStore(store =>
{
    store.RegisterServer<ApiServer>();

#if DEBUG
    // 生成客户端调用代码
    var codeString = store.GetProxyCodes("WebApiProxy", typeof(WebApiAttribute));
    File.WriteAllText("../../../WebApiProxy.cs", codeString);
#endif
});
```

### 7.2 使用生成的代理

生成的代理代码包含了所有 WebAPI 方法的客户端接口，可以直接使用。

### 7.3 源生成器

使用源生成器在编译时生成代理：

```csharp
[GenerateWebApiProxy]
public partial class ApiServer : SingletonRpcServer
{
    [WebApi(Method = HttpMethodType.Get)]
    public int Sum(int a, int b)
    {
        return a + b;
    }
}
```

生成的接口可以直接用于创建代理。

## 八、客户端插件

### 8.1 请求拦截

可以在发送请求前后进行拦截处理：

```csharp
private class MyWebApiPlugin : PluginBase, IWebApiRequestPlugin, IWebApiResponsePlugin
{
    public async Task OnWebApiRequest(IWebApiClientBase client, WebApiEventArgs e)
    {
        if (e.IsHttpMessage)
        {
            // 使用 System.Net.Http.HttpClient 时
            e.RequestMessage.Headers.Add("Token", "123456");
        }
        else
        {
            // 使用 TouchSocket HttpClient 时
            e.Request.Headers.Add("Token", "123456");
        }
        
        await e.InvokeNext();
    }

    public async Task OnWebApiResponse(IWebApiClientBase client, WebApiEventArgs e)
    {
        // 处理响应
        await e.InvokeNext();
    }
}
```

### 8.2 使用插件

```csharp
await client.SetupAsync(new TouchSocketConfig()
    .SetRemoteIPHost("127.0.0.1:7789")
    .ConfigurePlugins(a =>
    {
        a.Add<MyWebApiPlugin>();
    }));
```

## 九、完整示例

### 9.1 服务器端

```csharp
[GenerateWebApiProxy]
public partial class UserApiServer : SingletonRpcServer
{
    [WebApi(Method = HttpMethodType.Get)]
    public List<UserDto> GetUsers()
    {
        return new List<UserDto>
        {
            new UserDto { Id = 1, Name = "Alice", Age = 25 },
            new UserDto { Id = 2, Name = "Bob", Age = 30 }
        };
    }

    [WebApi(Method = HttpMethodType.Get)]
    public UserDto GetUserById(int id)
    {
        return new UserDto { Id = id, Name = $"User{id}", Age = 20 + id };
    }

    [WebApi(Method = HttpMethodType.Post)]
    public string CreateUser([FromBody] UserDto user)
    {
        return $"Created user: {user.Name}";
    }
}

public class UserDto
{
    public int Id { get; set; }
    public string Name { get; set; }
    public int Age { get; set; }
}
```

### 9.2 客户端调用（方式一：HttpClient）

```csharp
using var httpClient = new HttpClient();
var json = await httpClient.GetStringAsync("http://localhost:7789/userapiserver/getusers");
var users = JsonConvert.DeserializeObject<List<UserDto>>(json);
```

### 9.3 客户端调用（方式二：WebApiClient）

```csharp
var client = new WebApiClient();
await client.SetupAsync(new TouchSocketConfig()
    .SetRemoteIPHost("127.0.0.1:7789"));
await client.ConnectAsync();

var request = new WebApiRequest();
request.Method = HttpMethodType.Get;

var users = await client.InvokeTAsync<List<UserDto>>("/userapiserver/getusers", null, request);
```

### 9.4 客户端调用（方式三：DispatchProxy）

```csharp
var httpClient = new HttpClient();
httpClient.BaseAddress = new Uri("http://localhost:7789");

var proxy = httpClient.GetWebApiDispatchProxy<IUserApiServerClient>();

var allUsers = await proxy.GetUsers();
var specificUser = await proxy.GetUserById(1);
var createResult = await proxy.CreateUser(new UserDto { Name = "Charlie", Age = 35 });
```

## 十、最佳实践

### 10.1 复用 HttpClient

```csharp
// 推荐：复用实例
private static readonly HttpClient _sharedClient = new HttpClient();

var result1 = await _sharedClient.GetStringAsync("...");
var result2 = await _sharedClient.GetStringAsync("...");
```

### 10.2 设置超时

```csharp
var client = new HttpClient();
client.Timeout = TimeSpan.FromSeconds(30);
```

### 10.3 使用连接池

```csharp
var client = new HttpClient();
client.BaseAddress = new Uri("http://localhost:7789");

var proxy = client.GetWebApiDispatchProxy<IApiServer>();

// 复用代理实例
var result1 = await proxy.Sum(1, 2);
var result2 = await proxy.Sum(3, 4);
```

### 10.4 异常处理

```csharp
try
{
    var result = await client.InvokeTAsync<int>("/apiserver/sum", null, request);
}
catch (TimeoutException)
{
    Console.WriteLine("请求超时");
}
catch (Exception ex)
{
    Console.WriteLine($"请求失败: {ex.Message}");
}
```
